Challenge

Este projeto foi desenvolvido utilizando Angular 19 e está hospedado em https://out-challenge.vercel.app/.

Requisitos

Antes de começar, certifique-se de ter os seguintes requisitos instalados em sua máquina:

  • Node.js (versão 18.19.1 ou superior)
  • Angular CLI (versão 19.2.5 ou superior)
  • Um navegador web moderno

Passo a passo para rodar o projeto

  1. Clone o repositório:
Example :
git clone https://github.com/gabrielpapke/out-challenge.git
cd out-challenge
  1. Instale as dependências:
Example :
npm install
  1. Inicie o servidor de desenvolvimento:
Example :
npm start

O aplicativo estará disponível em http://localhost:4200/.

  1. Execute os testes unitários:
Example :
npm run test

Isso executará os testes via Karma.

  1. Gere a build de produção:
Example :
npm run build

Os artefatos da build serão armazenados no diretório dist/.
